# Учебная проектная работа - MESTO-REACT-AUTH

## Описание
Данный проект - продолжение Учебной проектной работы №3 - MESTO и №4 - MESTO-REACT.  Дополнительно реализована авторизация и регистрация пользователя. Проект взаимодействует сразу с двумя API: авторизации и API редактирования пользователя и получения карточек.  Реализована работа с локальным хранилищем и токеном. По макету из FIGMы свёвстаны новые компоненты (Login, Register, ProtectedRoute, InfoTooltip). Основная функциональность приложения подключена бэкэнду Яндекс.Практикума. Функциональность приложения доступна только авторизованным пользователям. Проект размещён на [GH PAGEs](https://miardo.github.io/react-mesto-auth/).

## Стек технологий
HTML, CSS, БЭМ, REACT
